Usage Guide

Follow these simple steps to get started

Installation

  • Download from WordPress plugin repository
  • Upload to your plugins directory
  • Activate through WordPress admin

Configuration

  • Go to Settings > Auto Link in WordPress admin
  • Configure global settings like case sensitivity and max links
  • Add keywords and corresponding URLs

Usage

  • Post tags will be automatically converted into links
  • View custom keywords in admin panel
  • Adjust settings as needed

FAQ

Answers to common questions

Why aren't some keywords automatically linked?

Please check whether the maximum number of links has been exceeded and whether the case sensitivity settings are met. In some cases, keywords inside HTML tags or within existing links will not be replaced.

How to disable linking in specific articles?

Simply add the excluded posts in the Auto Link settings in the admin panel.
