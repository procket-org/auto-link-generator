# 自動リンク生成プラグイン

**自動リンク生成プラグイン**は、WordPressの投稿内のタグやキーワードを自動的に内部/外部リンクに変換し、SEO対策とユーザー体験を向上させるプラグインです。

---

## ✨ 特徴

- 🌐 多言語対応（中国語、日本語、韓国語など）
- 🏷️ 投稿タグを自動的にキーワードとして使用
- 🔗 キーワードを自動リンクに変換
- 🔍 大文字小文字を区別するオプション
- ⚙️ 1投稿あたりのリンク数制限
- 🚫 特定の投稿を除外可能
- 🖥️ Vue.js製の直感的な管理画面
- 🔄 WordPress REST API完全対応

---

## 🧩 インストール方法

1. プラグインZIPファイルをダウンロード
2. WordPress管理画面で **プラグイン > 新規追加 > アップロード**
3. ZIPファイルをアップロード後、有効化

---

## ⚙️ 使い方

### 基本設定

1. **設定 > 自動リンク** に移動
2. 以下の設定を行います：
    - **タグ自動取得**：有効時、タグを自動的にキーワードとして使用
    - **大文字小文字区別**：有効時、完全一致が必要
    - **最大リンク数**：1投稿あたりのリンク上限（0=無制限）

### 投稿の除外

- 自動リンクを適用したくない投稿を選択
- 除外リストは随時編集可能

### キーワード管理

- リンク変換したいキーワードとURLを登録
- 編集、削除、一括操作が可能

---

## ⚙️ 動作原理

プラグインが投稿内容をスキャンし、定義済みキーワードを検出してリンクに変換します。HTML構造を保持し、コードブロック等には影響しません。

---

## 🔧 技術詳細

- **フロントエンド**：Vue.js + Axios
- **バックエンド**：WordPress REST API
- **データ保存**：WordPress Options API

---

## ⚠️ 注意事項

- リンクが多すぎるとユーザー体験が低下する可能性あり
- 重要な記事は除外リストに追加を推奨

---

## ❓ よくある質問

**Q: 特定キーワードの自動リンクを無効にするには？**  
A: キーワード管理ページで該当キーワードを削除してください。

**Q: 一部のキーワードがリンク化されない理由は？**  
A: 以下の可能性があります：
- 投稿が除外リストに入っている
- リンク上限に達している
- HTMLタグ内やコードブロック中のキーワード

**Q: タグがアーカイブページにリンクするのを防ぐには？**  
A: カスタムキーワードが優先されます。タグと同じ名前のキーワードを登録し、希望のURLを設定してください。
